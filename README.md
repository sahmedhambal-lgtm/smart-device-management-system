# Smart Device Management System

## Project Description

The Smart Device Management System is a Python Object-Oriented Programming project developed to demonstrate the concepts of classes, objects, inheritance, encapsulation, constructors, getters and setters, and menu-driven programming.This project will manage smart devices used in smart homes.

### The system manages three different smart devices used in a smart home:

1. Temperature Sensor
2.Smart Light
3.Security Camera

Each device inherits common features from the `SmartDevice` which is parent class while also having its own unique attributes and methods.

## Methods under SmartDevice (Parent Class)

1.Turn device on

2.Turn device off

3.Display device information

4.Get and set device ID

5.Get and set power status

## Methods and attribute under Temperature Sensor(child class)

1.Read temperature

2.Store temperature value

## Methods under SmartLight(child class)

1.Increase brightness

2.Decrease brightness

3.Brightness validation(from 0 to 100)

## Methods under Security Camera

1.Start recording

2.Stop recording

## Menu-Driven Interface

The program provides a simple menu that allows the user to:

1. Display Device Information

2. Turn On Devices

3. Turn Off Devices

4. Read Temperature

5. Adjust Brightness

6. Start Recording

7. Stop Recording

8. Exit
